1. 데이터 삭제

데이터를 삭제하거나 맵핑을 제거하는 등의 작업은 해당 버튼을 클릭 후 Confirm 버튼을 한 번 더 클릭해야 실제 작업이 수행됩니다.

스페이스 삭제, 어플리케이션 삭제, 서비스 Unbind 등의 대부분의 Delete 작업이 이에 해당됩니다.

2. 조직, 스페이스, 어플리케이션의 이름 변경

조직과 스페이스, 어플리케이션의 이름은 해당 텍스트에 마우스를 가져가면 연필모양이 표시되며 클릭시 편집모드로 전환됩니다.

이름을 변경한 후 enter키를 누르거나 다른 부분을 클릭하면 변경 작업이 수행됩니다.

3. 어플리케이션, 서비스 인스턴스의 상세 화면

메인 화면의 어플리케이션 목록이나 상단 메뉴 Service Instances 화면의 서비스 인스턴스 목록은 각 Row를 클릭하면 해당 Row의 상세정보 화면이 하단에 나타납니다.

기본적으로 최상단의 Row가 자동으로 선택되게 됩니다.

4. 멤버별 권한 관리

상단 메뉴의 Members 화면에서는 멤버별로 조직과 스페이스의 권한을 관리할 수 있습니다.

조직을 선택하고 스페이스를 선택하지 않을 경우(Select a space로 선택되어진 경우) 멤버별 조직 권한 관리 화면이 나타나고 스페이스를 선택할 경우 멤버별 해당 스페이스의 권한 관리 화면이 나타납니다.

삭제는 멤버가 삭제되는것이 아닌(사용자 계정이 삭제되는것이 아님), 해당 조직이나 스페이스에서 모든 권한을 잃게 되는 기능입니다.

화면 하단의 Invite 기능을 이용하면 특정 사용자를 현재 선택된 조직에 초대할 수 있습니다. 초대된 멤버는 해당 조직의 Auditor 권한이 주어집니다. 만약 생성되지 않은 사용자라면 초기 비밀번호가 1111로 세팅됩니다.
